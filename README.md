## Description

Backend API for [eShelf](https://github.com/levanvux/eshelf) - an online reading platform powered by [Nest](https://github.com/nestjs/nest) framework.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
